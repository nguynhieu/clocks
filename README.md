# Clocks

Clocks is web used to view clocks of different time zones

## Demo

[https://clocks-2c5ec.web.app/](https://clocks-2c5ec.web.app/)

## Installation

First

```bash
git clone https://github.com/nguynhieu/clocks.git
```

Then

```bash
npm install
```

Then

```bash
npm start
```
